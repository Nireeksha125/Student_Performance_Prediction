# Student Performance Prediction System

A Machine Learning based web application that predicts student performance using Math, Reading, and Writing scores.

## Features
- Predict student performance
- Dashboard to view stored predictions
- Edit and delete records
- Filter by Good / Average / Poor
- Dark & Light theme toggle
- SQLite database integration
- Responsive UI

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Scikit-learn
- Pandas

## Machine Learning Model
- Random Forest Classifier

## Project Workflow
User Input → Flask Backend → ML Model Prediction → Store Data in SQLite Database → Display in Dashboard

## Installation

Clone repository:
```bash
git clone https://github.com/Nireeksha125/Student_Performance_Prediction.git
```

Install dependencies:
```bash
pip install -r requirements.txt
```

Run project:
```bash
python app.py
```
